## Technical Setup for Recording

### Before Recording:
1. **Clean test account**: Create a fresh Google account with sample family photos
2. **Sample data**: Pre-populate with a small family tree (3-4 people)
3. **Sample photos**: Add 5-10 appropriate family-style photos to Google Photos
4. **Clear browser**: Use incognito mode or clear cache
5. **Screen resolution**: 1920x1080 for clarity
6. **Audio**: Use quality microphone, quiet environment

### Recording Tips:
1. **Slow down**: Move cursor slowly and deliberately
2. **Highlight clicks**: Use a screen recording tool that shows clicks
3. **Pause on important screens**: Hold on consent screens for 2-3 seconds
4. **Show URLs**: Make sure the browser address bar is visible
5. **No personal data**: Use obviously fake names (John Smith, Jane Doe, etc.)

### Editing Checklist:
- [ ] Add title card at beginning
- [ ] Add text overlays for key points
- [ ] Blur any sensitive information
- [ ] Add background music (soft, professional)
- [ ] Add captions/subtitles
- [ ] Export in 1080p
- [ ] Keep under 5 minutes
