# Project Overview: Roots & Branches

## Purpose
"Roots & Branches" is a comprehensive family tree application designed to help users visualize their lineage, document family history, and discover connections through media and location data. It emphasizes a modern, interactive user experience with features like deep Google Photos integration, interactive maps, and rich storytelling.

## Core Value Proposition
-   **Visual Discovery**: Moving beyond static charts to interactive graphs and maps.
-   **Media Integraiton**: Seamlessly connecting faces and places via Google Photos.
-   **Privacy & Control**: Granular permissions (Owner/Editor/Viewer) and data ownership.

## Current State (as of Oct 2023)
-   **Phase**: Post-MVP Polish & Feature Expansion.
-   **Active Development**: Email Notification System (`feat/notifications-and-prompts`).
-   **Production Status**: Live on Vercel (Frontend) and Render (Backend).
-   **Domain**: `familytree-e.com` (GoDaddy).

## Key Terms
-   **Tree**: The central container for a lineage.
-   **Person**: A node in the tree.
-   **Relationship**: The edge connecting nodes (Parent/Child, Spouse).
-   **Dual OAuth**: The architecture splitting Login from Google Service integration.
-   **Magic Link**: Passwordless email login method.
